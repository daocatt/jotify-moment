# Telegram Bot 命令文档

## 绑定账户

### `/start <token>`

将 Telegram 账户与 Moment 账户绑定。

**获取绑定 Token：**
1. 登录 Moment，进入个人主页
2. 点击编辑资料，找到 "Telegram" 选项
3. 点击生成绑定 Token
4. 将 Token 发送给 Bot：`/start abc123def456`

**示例：**
```
/start a1b2c3d4e5f6
```

**成功回复：**
```
🎉 绑定成功！
现在你可以向我发送文字、图片、语音或视频，它们将自动发布到你的 Moment 中。

发送 /help 查看使用指南。
```

**失败回复：**
```
❌ 绑定失败：未找到该绑定 Token，或该 Token 已失效。
请在 Moment 个人主页重新生成绑定 Token。
```

### `/start`（无参数）

- **已绑定用户**：回复确认绑定状态 + 使用指南
- **未绑定用户**：回复绑定操作指引 + 使用指南

## 帮助与个人信息

### `/help`

查看 Bot 使用指南。

**回复：**
```
📖 Moment Bot 使用指南

🔹 发帖：直接发送文字、图片、语音、视频（或组合发送）
🔹 绑定：在 Moment 个人主页的资料编辑中生成绑定 Token，然后发送 /start <token>
🔹 个人信息：/me
🔹 帮助：/help

📌 多张图片请作为相册发送，会合并到一条动态中
📌 管理员发帖直接通过，普通用户可能需要审核
```

### `/me`

查看当前绑定的个人注册信息及 Moment 主页地址（仅限私聊）。

**已绑定用户回复示例：**
```
👤 你的个人注册信息：

昵称：张三
邮箱：zhangsan@example.com
角色：普通用户 (user)
状态：正常
注册时间：2026/7/6 23:27:11

🏠 个人主页地址：
https://your-moment-domain.com/u/zhangsan
```

**未绑定用户回复：**
```
⚠️ 你还没有绑定 Moment 账户。
请先在 Moment 个人主页生成绑定 Token，然后发送 /start <token>
```

## 发帖

绑定后，直接向 Bot 发送消息即可发帖。支持以下类型：

### 文字

直接发送文字消息，内容作为动态正文。

```
今天天气真好！
```

### 图片

发送图片，自动上传到 Moment 存储。

- 单张图片：直接发送
- **多张图片（相册）**：以相册形式发送多张图片，会合并到同一条动态中
- 支持 JPG、PNG、GIF、WebP 格式
- 图片会自动生成 400×400 缩略图用于列表展示，点击可查看原图

### 语音

发送语音消息（按住麦克风录制），自动上传为音频附件。

- 格式：OGG/Opus（Telegram 默认）
- 时长信息会保留

### 视频

发送视频消息，自动上传为视频附件。

- 支持 MP4、WebM 格式
- 视频笔记（圆形视频）同样支持
- 时长信息会保留

### 音频文件

以文件形式发送音频（非语音消息），支持 MP3、WAV、OGG、M4A、FLAC。

### 文档

以文件形式发送的图片/音频/视频文档，Bot 会根据 MIME 类型和扩展名自动识别并处理。

### 组合发送

图片 + 文字：发送带 caption 的图片，caption 作为动态正文。

```
[图片]
caption: 在公园拍到的夕阳 🌅
```

### YouTube 链接

文字中包含 YouTube 链接时，自动提取视频 ID 嵌入动态。

## 审核机制

- **管理员**（super_admin / admin）：发帖直接通过
  - Bot 回复：`✅ 动态已发布 (N 个附件)！` 或 `✅ 动态已发布！`
- **普通用户**：
  - 若系统开启"普通用户发帖需要审核"，动态进入待审核状态
  - Bot 回复：`📝 动态已提交，等待管理员审核后发布。`
  - 若未开启审核，直接发布，Bot 回复同管理员

### 相册动态回复

- 审核中：`📝 相册动态已提交（N 张图片），等待管理员审核。`
- 已发布：`✅ 相册动态已发布（N 张图片）！`

## 错误处理

| 场景 | 回复 |
|------|------|
| 未绑定用户发消息 | `⚠️ 你还没有绑定 Moment 账户。请先在 Moment 个人主页生成绑定 Token，然后发送 /start <token>` |
| 发送空消息（无文字无媒体） | `💡 请发送文字、图片、语音或视频来发布动态。发送 /help 查看使用指南。` |
| 绑定 Token 无效/过期 | `❌ 绑定失败：未找到该绑定 Token，或该 Token 已失效。请在 Moment 个人主页重新生成绑定 Token。` |
| 发送未知命令（`/`开头） | `❌ 未知命令。发送 /help 查看可用命令。` |

## 技术细节

### 相册（Media Group）合并

Telegram 发送多张图片时，会将每张图片作为独立的 webhook 消息发送，但所有消息共享相同的 `media_group_id`。Bot 使用内存缓存 + 2 秒窗口期收集同一 `media_group_id` 的所有消息，然后合并为一条动态发布。

### 文件上传

所有媒体文件通过 Moment 统一上传接口处理：
- 根据后台配置选择本地存储或 S3/R2 云存储
- 图片自动生成 400×400 缩略图
- 文件名使用 32 位随机加密哈希，不含原始文件名
- 路径按 `YYYYMM/` 时间目录切分
- 受后台配置的文件大小限制和允许后缀限制

### MIME 类型推断

Telegram 文件的 MIME 类型推断采用多层策略：

1. **扩展名映射**：根据文件扩展名（jpg/png/ogg/mp4 等）直接映射
2. **路径关键词 fallback**：当扩展名未知导致 MIME 为 `application/octet-stream` 时，根据 Telegram 文件路径中的关键词推断：
   - `voice` 路径 → `audio/ogg`
   - `audio` 路径 → `audio/mpeg`
   - `photo` 路径 → `image/jpeg`
   - `video` 路径 → `video/mp4`
3. **调用方二次校验**：各消息类型处理器在 MIME 仍不正确时，基于文件扩展名做最终 fallback：
   - voice → `audio/ogg`
   - audio → 根据扩展名选择 `audio/ogg`/`audio/mpeg`/`audio/wav`/`audio/mp4`，默认 `audio/mpeg`
   - video → 根据扩展名选择 `video/webm`，默认 `video/mp4`
